VinylParadise

VinylParadise est une application mobile Android que j'ai développée dans le cadre d'un de mes cours. Elle est dédiée aux passionnés de vinyles et permet aux utilisateurs d'acheter et de vendre des vinyles, de rechercher des albums à l'aide de filtres personnalisés. Avec une interface intuitive , VinylParadise offre une expérience fluide pour les collectionneurs et amateurs de musique vintage.

API VinylParadise

L'API VinylParadise est une interface de programmation que j'ai développée dans le cadre d'un de mes cours pour soutenir les fonctionnalités de l'application mobile VinylParadise. Cette API REST permet la gestion des utilisateurs, des annonces de vinyles, des recherches avec filtres. Elle offre des points de terminaison sécurisés et performants pour effectuer des opérations CRUD (Créer, Lire, Mettre à jour, Supprimer) sur les données, garantissant ainsi une intégration fluide et une expérience utilisateur optimale.
